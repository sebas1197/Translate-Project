# Transalate project

- El proyecto lista palabras de inglés con su traducción y pronunciación.

## Pre-requisitos 📋

- Internet.
- Navegador web actual.

## Ejecución 🛠️

- Permite escuchar la pronunciación de una palabra en específico o todo el listado con su traducción.
